# Cara Berkontribusi di Thriftopia

Makasih udah tertarik buat kontribusi di Thriftopia! Ini panduan singkat cara berkontribusinya.

## Kode Etik

Dengan ikut berkontribusi, kamu setuju buat mengikuti Kode Etik kita ya. Intinya, saling menghargai aja.

## Cara Kontribusi

1. Fork dulu repository-nya
2. Clone fork-anmu:
```bash
git clone https://github.com/username-kamu/thriftopia.git
```

3. Bikin branch baru:
```bash
git checkout -b fitur/nama-fiturmu
```

4. Ubah kodenya terus commit:
```bash
git add .
git commit -m "feat: nambahin fitur apa gitu"
```

5. Push ke fork-mu:
```bash
git push origin fitur/nama-fiturmu
```

6. Bikin Pull Request deh

## Aturan Commit Message

Kita pakai [Conventional Commits](https://www.conventionalcommits.org/). Contohnya gini:

- `feat:` - Fitur baru
- `fix:` - Benerin bug
- `docs:` - Update dokumentasi
- `style:` - Perubahan format kode
- `refactor:` - Bersihin/improve kode
- `test:` - Nambah atau benerin test
- `chore:` - Update tools atau config

## Setup Development

Buat setup project, bisa cek [README.md](README.md) ya.

## Proses Pull Request

1. Update README.md kalo perlu
2. Ikutin style coding yang udah ada
3. Jangan lupa tambahin test
4. Pastiin semua test passed
5. Update dokumentasi kalo perlu

## Ada Pertanyaan?

Santai aja, bisa langsung bikin issue di repository kalo ada yang mau ditanyain.

## Lisensi

Dengan berkontribusi di Thriftopia, kamu setuju kalo kontribusimu akan dilindungi lisensi MIT ya.
